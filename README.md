# JustAChatBot
This is just a discord bot


### Prerequisites

```
Node v6.0+
Nodemon (optional)
```

### Installing

```
git clone https://github.com/drrr-devs/justachatbot.git
git cd https://github.com/drrr-devs/justachatbot.git
npm install
nodemon server.js
```
